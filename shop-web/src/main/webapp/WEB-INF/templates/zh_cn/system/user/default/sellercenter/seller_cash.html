提现说明
1.

单笔提现金额最低100元，限100的整数倍提取，最高限1万元；单次提现1万元时，工作人员需电话核实，请保持手机畅通；

2.

使用支付宝或网银提现时，请确认收款名与认证名一致；否则将导致提现不成功；

3.

银行卡充值后提现时，请保持提现的账户与充值的账户一致，否则将导致提现不成功；

4.

本周工作日内的提现申请，于下周工作日到账，所有法定节假日（含周末）均不处理提现；

5.

请尽量使用支付宝提现；长假等因素导致提现业务量大或其它情况时，到账均有延迟；

6.

严禁使用信用卡套现、虚假交易等行为，一经发现，将对该帐户进行冻结处理；

7.

如遇银行系统、微信和支付宝系统升级等特殊原因，提现金额可能会延迟到帐。
