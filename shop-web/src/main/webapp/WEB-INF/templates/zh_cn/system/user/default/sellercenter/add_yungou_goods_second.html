基本信息
*商品分类： $!goods_class_info
*商品名称： 商品标题名称长度至少3个字符，最长50个汉字
*(零售价)零售价： 元 零售价必须是0.01~1000000之间的数字
商品类型： 充值卡、消费券等属于虚拟商品
货到付款： 该商品是否支持货到付款
增值税发票： 该商品是否可以开具增值税发票
*(会员价格)供货价： 元
  • 供货价必须是0.01~1000000之间的数字
  • 若启用了库存配置，请在下方商品库存区域进行管理
  • 商品规格库存表中如有价格差异，供货价显示为价格区间形式
*需要的重消： 元
  • 重消必须是0~1000000之间的数字
*需要的积分： 积分 （供货价的%）
  • 积分必须为整数
  • 积分必须是0~1000000之间的数字
  • 普通会员以积分+现金购买，CN会员以购物积分+现金购买
 *需要的现金： 元 （供货价的%）
  • 现金必须是0~1000000之间的数字
