API文档 个人o2o ShopSearchGoods

ShopSearchGoods

简要描述

搜索商品列表

属性

名称 类型 描述
id Long 商品id
goodsName String 商品名称
goodsMainPhotoPath String 商品主图片
hasExchangeIntegral Boolean 是否有积分兑换价格
goodsIntegralPrice BigDecimal 商品积分兑换后的价格，无兑换显示原价
hasMobilePrice Boolean 是否有手机专享价
goodsMobilePrice BigDecimal 手机专享价，如果不设置手机专享价则留空
goodsCurrentPrice BigDecimal 商品当前价格
exchangeIntegral Integer 兑换积分值
goodsShowPrice BigDecimal 商品显示价格
mobilePrice BigDecimal 手机专享价
